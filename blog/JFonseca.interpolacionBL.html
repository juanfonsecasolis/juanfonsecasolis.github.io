Interpolación de banda limitada para el tipo de cambio del dólar¶

2018 Juan M. Fonseca-Sols

Resumen¶

La interpolacin de banda limitada es un mtodo numrico que permite reconstruir seales continuas a partir de muestras tomadas de manera irregular, es decir, sin seguir una tasa de muestreo fija. Este es un problema interesante de resolver para cualquier investigador que se encuentre estudiando una seal construida con datos dispersos, pues le permite alcanzar una buena aproximacin an cuando los datos no cumplan las reglas de muestreo de Nyquist (un problema frecuente para los ingenierios de minado de datos). Para ejemplificar la utilidad del mtodo se propone estudiar el comportamiento del tipo de cambio del dlar en Costa Rica desde el 2014, un tema que se a puesto de moda ya que el 4 de setiembre del 2018 la moneda subi al rango de 580 a 585 colones, afectando a gente que tena deudas en esa moneda por el incremento en las tasas de inters. El Banco Central de Costa Rica (BCCR) provee un servicio web que permite obtener los datos de manera equidistante, pero se realiza un muestreo aleatorio sobre esa informacin para simular el caso de un investigador que solo cuente con anotaciones propias realizadas de manera irregular.

Interpolación de banda limitada¶

Ahora usamos interpolacin de banda limitada para aproximar la seal original usando series de Fourier, es decir haciendo una suma de senoidales con una amplitud, frecuencia y desfase distintos.4

In [193]:
from numpy import *
from matplotlib.pylab import *
import random

def BL_interp_1D(x, z, T, order, grid_step=0.01, win=True):
    """
    Band-limited interpolation of 1D functions (cdigo obtenido de P.Prandoni, M.Vetterli)
    """
    # Create Fourier order vector
    k = expand_dims(arange(-order, order+1), 0)
    
    # construct the Fourier matrix
    F = exp(2j*pi*x*k/(T[1]-T[0]))
    
    # Least-square projection (alternatively linalg.lstsq can be used)
    C = dot(dot(linalg.inv(dot(F.T,F)), F.T), z)
    
    # create new evenly spaced grid
    xg = expand_dims(arange(T[0], T[1], grid_step), 1)
    
    # window the Fourier coefficients if requested
    if (win):
        C *= expand_dims(hanning(2*order+1), 1)
    
    zg = dot(exp(2j*pi*xg*k/(T[1]-T[0])), C)
        
    return zg, xg, C, k

Aplicamos la interpolacin BL
'''
orden = 23 # nmero de coeficientes de la aproximacin (emprico)

Como podemos apreciar, es posible obtener una mejor aproximacin (que usando interpolacin lineal) con unos pocos coeficientes, es ms si la seal es peridica el mtodo BL puede funcionar incluso si el muestreo de hace de forma aleatoria.

En contrapartida, si la seal es aperidica y se pide realizar una aproximacin entre dos puntos lejanos, el mtodo BL podra agregar saltos donde no los hay. Sin embargo, es aceptable porque el mtodo usa pocos coeficientes y porque para esos casos ya existe la interpolacin spline.

Cantidad de coeficientes¶

Como la grfica de los coeficientes es par, solo necesitamos almacenar la mitad de los coeficientes, es decir que podemos calcular el porcentaje de compresin de la siguiente manera:

$$\text{nc}=\frac{dim(\text{zm})-0.5 \, \text{orden}}{dim(\text{zm})}$$
