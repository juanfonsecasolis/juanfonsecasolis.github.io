Interpolación de banda limitada para el tipo de cambio del dólar¶

2018 Juan M. Fonseca-Sols

Resumen¶

La reconstruccin de secuencias temporales continuas a partir de muestras tomadas discretamente es un problema que ha sido resuelto tradicionalmente usando el teorema de Nyquist. Este teorema establece que la frecuencia de muestreo debe ser el doble de la frecuencia mxima contenida en la seal original para evitar la prdida de informacin. La reconstruccin –tambin llamada interpolacin– se logra convolucionando una funcin de interpolacin –tambin llamada ncleo de interpolacin– con cada muestra disponible, luego se realiza una suma de todas las convoluciones para obtener la funcin continua. El problema se complica cuando se dispone de muestras tomadas a una frecuencia de muestreo irregular, donde, en lugar de frecuencia de muestreo, se habla de una tasa de innovacin (FIR, por sus siglas en ingls). En este caso existe un mtodo llamado interpolacin de banda limitada (BLI, por sus siglas en ingls) que utiliza las series de Fourier y el mtodo de mnimos cuadrados para producir una aproximacin de la seal original. El mtodo BLI supone que la seal a representar no tiene frecuencias altas, pero si las tiene, basta con aplicar un filtrado pasabajas. En este notebook se explica con mayor detalle en qu consiste el mtodo mencionado y se presenta un ejemplo procesando el tipo de cambio del dlar en Costa Rica ofrecido por el Banco Central de Costa Rica (BCCR).

Decimación aleatoria de la señal original¶

Para demostrar la utilidad de la BLI simulamos el caso en el que se cuenta con muestras tomadas a intervalos aleatorios. Para ello, iteramos por cada entrada de la seal original y la conservamos si el nmero aleatorio generado es menor al valor $p$.

In [8]:
import random
p = 0.2 # probabilidad de conservar una muestra

xm = []
zm = []
for dia in range(len(x)):
    if random.random()<p:
        xm.append(x[dia])
        zm.append(z[dia])

Interpolación de banda limitada¶

Ahora usamos BLI para aproximar la seal original usando una curva ms suave.4

In [10]:
def BL_interp_1D(x, z, T, order, grid_step=0.01, win=True):
    """
    Band-limited interpolation of 1D functions (P.Prandoni, M.Vetterli)
    """
    # Create Fourier order vector
    k = np.expand_dims(arange(-order, order+1), 0)

    # construct the Fourier matrix
    F = exp(2j*pi*x*k/(T[1]-T[0]))

    # Least-square projection (alternatively linalg.lstsq can be used)
    C = dot(dot(linalg.inv(dot(F.T,F)), F.T), z)

    # create new evenly spaced grid
    xg = expand_dims(arange(T[0], T[1], grid_step), 1)

    # window the Fourier coefficients if requested
    if (win):
        C *= expand_dims(hanning(2*order+1), 1)

    zg = dot(exp(2j*pi*xg*k/(T[1]-T[0])), C)

    return zg, xg, C, k

'''
Aplicamos la interpolacin BL
'''
orden = 20 # nmero de coeficientes de la aproximacin (emprico)
T = [0, N]
xmp = expand_dims(array(xm),1)
zmp = expand_dims(array(zm),1)

# calculamos la interpolacin BL
zp1, xp1, C1, k1 = BL_interp_1D(xmp, zmp, T, orden, win=False)

Cantidad de coeficientes¶

Como la grfica de los coeficientes es par, solo necesitamos almacenar la mitad de los coeficientes, es decir que podemos calcular el porcentaje de compresin de la siguiente manera:

$$\text{nc}=\frac{dim(\text{zm})-0.5 \, \text{orden}}{dim(\text{zm})}$$
In [12]:
nc = (len(zm)-(orden*0.5))/len(zm)
print 'Muestras tomadas: ',len(zm)
print 'Cantidad de coeficientes: ', orden/2, ' (necesarios para reproducir la seal)'
print 'Compresin lograda: ', int(nc*100),'%'
